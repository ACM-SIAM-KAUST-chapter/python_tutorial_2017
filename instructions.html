Setup instructions

We recommend you install the all-in-one scientific Python distribution from Continuum Analytics (Anaconda). However, you can install all the necessary packages and their dependencies on your own using pip or easy_install installers.

A) You can install Anaconda by doing the following:

  1. Download the installer that matches your operating system and follow the instructions. You can choose between the Graphical Installer or the Command Line Installer.
  2. You are now ready to use Python. To test your installation open a terminal and run 
    ipython
    You should see an interactive window that says you are using the Anaconda Python distribution. You can open a new notebook by typing 
    jupyter notebook
    Start programming!

B) Alternatively, you can install a light version of Anaconda: Miniconda.
First download the installer depending on your operating system from here. Then open a terminal and type the following: 

conda install numpy scipy matplotlib jupyter
This will install only the libraries necessary for this tutorial.

For any questions about the installation go to Anaconda documentation or email us at acm.chapter@kaust.edu.sa .
